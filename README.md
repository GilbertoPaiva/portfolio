# Portfolio

Portfólio pessoal desenvolvido com React, TypeScript e Styled Components. Este projeto tem como objetivo apresentar informações, projetos e habilidades de forma moderna e responsiva.

## Tecnologias

- [React](https://reactjs.org/) 19
- [TypeScript](https://www.typescriptlang.org/)
- [Styled Components](https://styled-components.com/)
- [Create React App](https://github.com/facebook/create-react-app)

## Como executar

```bash
# Instalar dependências
npm install

# Rodar em modo de desenvolvimento
npm start

# Executar testes
npm test

# Gerar build de produção
npm run build
```

O servidor de desenvolvimento estará disponível em [http://localhost:3000](http://localhost:3000).
